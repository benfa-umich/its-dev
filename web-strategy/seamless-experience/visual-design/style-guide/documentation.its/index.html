Typography

documentation.ITS uses typography to create hierarchies of content and to promote readability. Open Sans is the default font for headers and body copy.

Grid

The pages are based on a 12-column responsive grid system. For desktop view, documentation.ITS utilizes a two-column layout. First column is for content. The second column is reserved for the optional sidebar column containing the Navigation Menu.
